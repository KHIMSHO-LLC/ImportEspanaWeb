## ¿Qué son las Tablas del BOE?

Cada año, el Ministerio de Hacienda publica en el **Boletín Oficial del Estado (BOE)** las tablas de valoración de vehículos. Estas tablas asignan un **valor fiscal oficial** a cada marca, modelo y versión de vehículo vendido en España.

La última orden vigente es la **Orden HAC/1501/2025**, publicada en diciembre de 2025, que establece los valores para el año 2026.

## ¿Para qué sirve el Valor Fiscal?

El valor fiscal es la piedra angular de varios impuestos relacionados con vehículos en España:

- **Impuesto de Matriculación (IEDMT)**: Se calcula sobre el valor fiscal depreciado
- **Impuesto de Transmisiones Patrimoniales (ITP)**: Para ventas entre particulares
- **Impuesto de Sucesiones y Donaciones**: Cuando se hereda o recibe un vehículo

Es importante entender que el valor fiscal **no es el precio de mercado real**. Un coche puede venderse en 15.000€ en el mercado, pero su valor fiscal según el BOE puede ser 22.000€, o viceversa.

## ¿Cómo se estructura la tabla?

Cada entrada en la tabla del BOE contiene:

| Campo                   | Descripción                                     |
| ----------------------- | ----------------------------------------------- |
| **Marca**               | Fabricante del vehículo                         |
| **Modelo/Denominación** | Nombre completo del modelo y versión            |
| **Años**                | Período de fabricación                          |
| **Cilindrada (cc)**     | Centímetros cúbicos del motor                   |
| **Cilindros**           | Número de cilindros                             |
| **Combustible**         | Gasolina (G), Diésel (D), Eléctrico (Elc), etc. |
| **Potencia (kW/CV)**    | Potencia del motor                              |
| **Valor (€)**           | Precio de referencia fiscal                     |

### Ejemplo de entrada

```
Marca: MERCEDES-BENZ
Modelo: Clase C 200 CDI BlueEFFICIENCY
Años: 2014-2021
CC: 2143
Cilindros: 4
Combustible: D
KW: 100
CV: 136
Valor: 37.200€
```

## La Depreciación: De "Nuevo" a "Usado"

El BOE publica los valores como si el coche fuera **nuevo**. Para vehículos usados, Hacienda aplica una **tabla de depreciación oficial** que reduce el valor según la antigüedad.

### Tabla de Depreciación Oficial

| Antigüedad | % del Valor que Retiene | Ejemplo (Valor Nuevo: 40.000€) |
| ---------- | ----------------------- | ------------------------------ |
| < 1 año    | 100%                    | 40.000€                        |
| 1-2 años   | 84%                     | 33.600€                        |
| 2-3 años   | 67%                     | 26.800€                        |
| 3-4 años   | 56%                     | 22.400€                        |
| 4-5 años   | 47%                     | 18.800€                        |
| 5-6 años   | 39%                     | 15.600€                        |
| 6-7 años   | 34%                     | 13.600€                        |
| 7-8 años   | 28%                     | 11.200€                        |
| 8-9 años   | 24%                     | 9.600€                         |
| 9-10 años  | 19%                     | 7.600€                         |
| 10-11 años | 17%                     | 6.800€                         |
| 11-12 años | 13%                     | 5.200€                         |
| > 12 años  | 10%                     | 4.000€                         |

Como puedes ver, un coche pierde el **44% de su valor fiscal** en solo 4 años. Esto tiene un impacto directo en los impuestos que pagarás.

## Casos Especiales

### Vehículos no incluidos en las tablas

Si tu coche no aparece en las tablas del BOE (modelos muy raros, importaciones de marcas no comunes, etc.), Hacienda puede:

- Utilizar un **modelo comparable** como referencia
- Solicitar una **tasación pericial**
- Aplicar el **precio de factura** como base

### Equipamiento opcional

Las tablas del BOE **no contemplan** equipamiento opcional. El valor fiscal es el mismo para una versión básica que para una con todos los extras. Esto puede ser una ventaja si compras coches bien equipados.

### Vehículos con matrícula diplomática o militar

Tienen regímenes especiales de valoración y tributación.
