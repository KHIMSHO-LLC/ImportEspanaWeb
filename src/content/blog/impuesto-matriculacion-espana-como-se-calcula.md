## ¿Qué es el Impuesto de Matriculación?

El Impuesto Especial sobre Determinados Medios de Transporte (IEDMT), conocido popularmente como **Impuesto de Matriculación**, es un tributo que se aplica en España a la primera matriculación definitiva de vehículos.

Este impuesto afecta tanto a los **coches nuevos comprados en España** como a los **vehículos importados** que se matriculan por primera vez en territorio español.

## ¿Cómo se calcula?

El cálculo del impuesto se basa en dos factores fundamentales:

1. **Las emisiones de CO2 del vehículo** — Determina el tipo impositivo (porcentaje)
2. **El valor fiscal del vehículo** — La base sobre la que se aplica ese porcentaje

### Tramos de CO2 (vigentes en 2026)

| Tramo       | Emisiones CO2  | Tipo Impositivo |
| ----------- | -------------- | --------------- |
| **Tramo 1** | ≤ 120 g/km     | **0%** (Exento) |
| **Tramo 2** | 121 - 159 g/km | **4,75%**       |
| **Tramo 3** | 160 - 199 g/km | **9,75%**       |
| **Tramo 4** | ≥ 200 g/km     | **14,75%**      |

### Buenas noticias para vehículos ecológicos

Los **vehículos eléctricos puros** (BEV) y los **vehículos de hidrógeno** están **completamente exentos** del impuesto de matriculación, ya que sus emisiones en ciclo WLTP son 0 g/km.

Los **híbridos enchufables (PHEV)** con emisiones inferiores a 120 g/km también quedan exentos.

## El Valor Fiscal: Tablas del BOE

La Agencia Tributaria no utiliza el precio que tú pagas por el coche para calcular el impuesto. En su lugar, utiliza el **valor fiscal oficial** publicado anualmente en el **Boletín Oficial del Estado (BOE)**.

### ¿Qué es el valor fiscal?

Es el precio de mercado que Hacienda asigna a cada marca, modelo y versión de vehículo. Se actualiza cada año mediante una Orden Ministerial.

Para vehículos usados, se aplica una **tabla de depreciación** según la antigüedad:

| Antigüedad     | Porcentaje que retiene |
| -------------- | ---------------------- |
| Menos de 1 año | 100%                   |
| 1-2 años       | 84%                    |
| 2-3 años       | 67%                    |
| 3-4 años       | 56%                    |
| 4-5 años       | 47%                    |
| 5-6 años       | 39%                    |
| 6-7 años       | 34%                    |
| 7-8 años       | 28%                    |
| 8-9 años       | 24%                    |
| 9-10 años      | 19%                    |
| 10-11 años     | 17%                    |
| 11-12 años     | 13%                    |
| Más de 12 años | 10%                    |

### Ejemplo práctico

Supongamos que importas un **BMW Serie 3 320d** con las siguientes características:

- Valor fiscal BOE (nuevo): **42.500€**
- Antigüedad: **4 años**
- Emisiones CO2: **132 g/km**

**Cálculo:**

1. Valor fiscal depreciado: 42.500€ × 47% = **19.975€**
2. Tipo impositivo (132 g/km → Tramo 2): **4,75%**
3. Impuesto de Matriculación: 19.975€ × 4,75% = **948,81€**

## ITP: El Otro Impuesto que Podrías Pagar

Si compras el coche a un **particular** (no a un concesionario profesional), además del impuesto de matriculación deberás pagar el **Impuesto de Transmisiones Patrimoniales (ITP)**.

El tipo del ITP varía según la comunidad autónoma:

| Comunidad  | Tipo ITP |
| ---------- | -------- |
| Madrid     | 4%       |
| Cataluña   | 5%       |
| Andalucía  | 4%       |
| Valencia   | 6%       |
| País Vasco | 4%       |
| Galicia    | 8%       |

La base imponible del ITP es el mismo valor fiscal depreciado del BOE.

## ¿Cómo pagar el Impuesto de Matriculación?

1. Rellenar el **modelo 576** de la AEAT
2. Presentarlo telemáticamente o en la oficina de Hacienda
3. Realizar el pago antes de acudir a la DGT para la matriculación

## Consejos para Ahorrar

1. **Busca coches con bajas emisiones** — Por debajo de 120 g/km no pagas matriculación
2. **Considera un eléctrico o híbrido enchufable** — Exentos al 100%
3. **Si el coche es muy antiguo** (+12 años), el valor fiscal es solo el 10% del precio nuevo
4. **Compara precios entre comunidades** — Si tu ITP local es alto, podrías matricular en otra comunidad (con matices legales)
